# Real-Time Object Detection and Tracking System

This project implements a real-time object detection and tracking system using **YOLOv8** and a simplified **SORT tracking algorithm**.  
It can detect and track multiple objects from a webcam or video file and optionally save the processed output video.

- Displays class name, confidence score, and bounding boxes
- Object movement trails (toggle on/off)
- Works with webcam or video file
- Option to save output video
- Screenshot capture feature
- Simple keyboard controls
