## **Description**
This repository contains multiple AI-powered tools built with Python for real-time applications and productivity enhancement:  

1. **Object Tracker** – Real-time object detection and tracking system using YOLOv8 and a lightweight SORT-style tracker.
2. **Language Translator** – Text translator with multi-language support and optional text-to-speech using Google Translate.

These tools are designed to be beginner-friendly, easy to run, and extendable for custom applications.

---

## **Features**
### Object Tracker 
- Real-time object detection from webcam or video files
- Tracks objects with unique IDs
- Visual trails for objects’ movement
- Displays FPS and detected objects on screen

### Language Translator
- Translate text between over 100 languages
- Text-to-speech audio output
- Simple GUI interface via Streamlit
